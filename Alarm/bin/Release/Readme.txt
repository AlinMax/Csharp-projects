Alarm

Description:
Alarm is a freeware program that mainly consists of a countdown timer for alerting the user. It has the option to calculate the time between the current time and a target time.

Shortcuts:
[Enter] - Start/pause alarm
[CTRL C] - Calculate time
[CTRL T] - Tooltip mode on/off
[CTRL P] - Popup mode on/off
[CTRL S] - Sound on/off
[CTRL R] - Reset alarm
[Esc] - Minimize program
[~] - Close program
